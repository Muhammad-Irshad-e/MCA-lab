    CSS (Cascading Style Sheets)

    CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in HTML or XML, including SVG, MathML, and XHTML.

The initial motivation for CSS was to address the limitations of HTML in controlling the visual presentation of web documents. At the time, styling was either handled by individual browser implementations or through deprecated HTML tags like , which mixed content and presentation. Several other style sheet languages were proposed around the same period, but CSS gained prominence and became the standard. The World Wide Web Consortium (W3C) played a crucial role in its development and standardization.
